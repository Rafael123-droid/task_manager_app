# Task Manager App

This is a **Flutter mobile app** for managing tasks, connected to the Task Manager API.

## Features

- User login and registration
- View, add, update, and delete tasks
- Synchronizes with the backend API
- Works on Windows and mobile platforms

## Installation

1. Clone the repo:

```bash
git clone https://github.com/Rafael123-droid/task_manager_app.git

2. Navigate to the project folder:
    cd task_manager_app

3. Install dependencies:    
    flutter pub get

4. Run the app:
    flutter run
